# Contributing to pax8-marketplace-mcp-skill

Thank you for your interest in this project.

## Feedback and Issues

We welcome feedback, bug reports, and suggestions via GitHub Issues. If you've found a problem with the skill content or have a workflow idea, please open an issue describing:

- What you were trying to do
- What happened (or what's missing)
- What you expected instead

## Forking and Adaptation

This project is licensed under Apache 2.0. You are welcome to fork it and adapt it for your own use under the terms of that license.

## External Pull Requests

We are not currently accepting external pull requests. This may change in the future — watch the repository for updates.

## Security Issues

Do not report security vulnerabilities via GitHub Issues. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Code of Conduct

All participants are expected to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
