# pax8-marketplace-mcp-recipes

A collection of open-source [Agent Skills](https://agentskills.io) and workflow recipes for the Pax8 Marketplace MCP Server (`mcp.pax8.com`) — teach AI assistants to query partner data, run common account management workflows, and work effectively with the Pax8 ecosystem.

## Skills

| Skill | Description |
|---|---|
| [pax8-marketplace-mcp-skill](./pax8-marketplace-mcp-skill/) | Query companies, subscriptions, invoices, products, quotes, orders, and usage via the Pax8 MCP Server |

## What is an Agent Skill?

A skill is a Markdown file (`SKILL.md`) that gives an AI assistant domain knowledge, tool guidance, and workflow recipes for a specific task. Skills follow the [agentskills.io specification](https://agentskills.io/specification) and can be loaded into Claude, Cursor, VS Code Copilot, n8n, or any compatible agent client.

## Using a Skill

See the [Agent Skills specification](https://agentskills.io/specification) for how to load and use skills across supported AI clients.
